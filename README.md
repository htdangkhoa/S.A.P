## Building
Project requires:  
  - [Node.js](https://nodejs.org/) version 4 or higher
  - [Electron framework](http://electron.atom.io/)
```
$ git clone https://github.com/htdangkhoa/S.A.P.git
$ cd S.A.P
$ npm install && npm start
```
